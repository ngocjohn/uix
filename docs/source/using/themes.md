# Themes

!!! info "Theme variable"
    The theme MUST define a variable `uix-theme` which MUST have the same value as the name of the theme. For example:
    ```yaml
    my-awesome-theme:
        uix-theme: my-awesome-theme

        ... other theme variables go here ...
    ```

## Theme variables

- `uix-card`
- `uix-row`
- `uix-glance`
- `uix-badge`
- `uix-heading-badge`
- `uix-assist-chip`
- `uix-element`

- `uix-root`
- `uix-view`
- `uix-more-info`
- `uix-sidebar`
- `uix-config`
- `uix-panel-custom`
- `uix-top-app-bar-fixed`
- `uix-dialog`

Also `<any variable>-yaml`.

## Classes

Set a class with:
    ```yaml
    uix:
        class: red
    ```
